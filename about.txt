## Galago Audio Project – Repository Summary

### What this repository is about

This repository contains an **early-stage machine learning pipeline for automatic galago (bushbaby) species identification from audio recordings**.
The core idea is to take short audio clips (WAV files), extract features, and use a trained neural network to predict the most likely galago species based on vocalizations.

The project is exploratory and research-driven, aimed at:

* Testing whether species-level identification from calls is feasible
* Evaluating model performance across a limited but curated call library
* Building a foundation that could later support a simple app or batch-processing tool

The repository focuses on **inference, evaluation, and experimentation**, rather than a polished end-user application.

---

### What currently exists in the repo

The codebase includes:

**1. Batch prediction scripts**

* Scripts that run a trained neural network on folders of WAV files
* Output top-N predictions per file (e.g. top-6 or top-7 species)
* Variants include:

  * Full-clip predictions
  * Windowed / sliding-window predictions for longer recordings

**2. Evaluation utilities**

* Scripts to compute confusion matrices for trained models
* Separate evaluation logic for different class configurations (e.g. 6-species vs 7-species models)
* Tools to inspect where the model confuses species

**3. Result inspection & summarization**

* Scripts to:

  * Inspect prediction outputs
  * Count detections per species
  * Sanity-check model behavior across datasets

**4. Supporting artifacts**

* File lists for batch processing
* Intermediate outputs used for debugging and comparison
